# 高鐵時刻表查詢應用程式

這是一個用於查詢台灣高鐵時刻表的程式。您可以使用它來查找高鐵列車的時刻表訊息，包括車次號碼、到站時間和抵達時間。

## 功能

此程式提供以下功能：

- 查詢指定車站之間的高鐵時刻表。
- 顯示車次號碼、到站時間和抵達時間的詳細信息。
- 可以選擇不同的出發車站和目的車站，以查詢不同路線的時刻表。
